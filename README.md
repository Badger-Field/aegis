# Aegis

Kodi 21 (Omega) addon delivery for the Aegis build.

## Install

1. Kodi → Settings → File Manager → Add Source
2. Path: `https://badger-field.github.io/aegis/`  Name: `aegis`
3. Settings → Add-ons → Install from zip file → `aegis` → `zips/repository.aegis/`
4. Install from repository → Aegis Repository → install the addons you want.

Direct repository-addon link: <https://badger-field.github.io/aegis/zips/repository.aegis/repository.aegis-0.1.0.zip>

## Source

Built from the private source repo. This repo only carries the signed
release payload (`addons.xml` + `zips/`). Last 3 versions of
each addon are retained for rollback.
